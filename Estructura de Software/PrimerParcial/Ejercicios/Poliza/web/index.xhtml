Polizas

Una compañua de seguros para autos ofrece dos tipos de póliza. Cobertura amplia (A) y daños a terceros (B). Para el plan A, la cuota base es de $1,200 y para el B, de $950. A ambos planes se les carga 10% de costo a la persona que conduce tiene por hábito beber alcohol, 5% si utiliza lentes, 5% si padece alguna enfermedad - Como defiencia cardica o diabetes -, y si tiene más de 40 años, se le carga 20%, de lo contrario sólo 10%.

Todos los cargos se realizan sobre el costo base. Hacer una aplicación para determinar cuánto le cuesta a una persona contratar una poliza.
