Escuela Ronaldo Calles

Ejercicio

La escuela "Ronaldo Calles" ortogará un bono a sus profesores de acuerdo a su antiguedad. Hacer una aplicación web en JSF que solicite los siguientes datos por medio de un formulario:

  • Nombre del profesor
  • Institución
  • Municipio
  • Edad
  • Años de servicio

    • El bono se ortogará:

  • $1000 por cada año de servicio
  • Un bono extra de $5000 si es mayor a 50 años
  • Un bono extra de $1000 si su municipio es San Juan del Río o Tequisquiapan

Se deberá desplegar una página mostrando el premio para el maestro...
